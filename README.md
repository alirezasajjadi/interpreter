# interpreter
This is an interpreter for a custom programming language  

## Project Description
In this project we can use 3 commands:
- print
- for loop
- assignment

and we have these operation:
- divide(/)
- factorial(!)
- multiply(*)
- substraction(-)
- sum(+)
